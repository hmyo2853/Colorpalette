# Colorpalette

## 노마드코더 사이드프로젝트 1주차 컬러팔레트

#### App 컨셉

- 선택한 색상과 반대되는 보색을 출력해준다.
- 헥사코드 copy ok!

#### App 설계

1. 컬러칩 만들기
2. 컬러 선택시 선택한 색상과 반대되는 보색 출력
3. 컬러 헥사코드 출력 및 복사 버튼 생성

#### 진행사항
1. 주 타이틀 만들기 [ O ]
2. 컬러칩 레이아웃 만들기 [ O ]
3. 컬러칩에 각 추천 컬러 픽업해서 하드 코딩 [ O ]
4. 컬러칩에 맞춰서 하단 box 컬러 색상 변경 확인 [ O ]
5. 헥사코드로 텍스트까지 출력 확인 [ O ]
6. 복사 버튼 css 적용 [ O ]
7. 복사 버튼에 이벤트 걸기 [ O ]
8. 복사된 복사 값 확인 [ O ]
9. 보색 설명 script 추가 [ O ]
